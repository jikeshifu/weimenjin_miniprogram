# 微门禁开源后台

这是微门禁开源版本后台代码，配合根目录 `weimenjin_app` 小程序端使用。本次发布版本由旧开源版本同步升级整理而来。

## 安装

1. 将站点运行目录指向 `weimenjin_admin/public`。
2. 导入 `weimenjin_open.sql`。
3. 复制 `.env.example` 为 `.env`，配置数据库、站点域名和小程序参数。
4. 登录后台，在“应用配置”中维护业务参数并生成配置。

默认超级管理员：

```text
账号：admin
密码：WmjDemo2026
```

首次登录后请及时修改默认密码。

## 在线更新

后台“系统更新”菜单已内置开源演示平台更新源：

```text
https://demo.wmj.com.cn/updates/manifest.json
```

进入页面会自动检测更新。检测到新版本后，超级管理员点击“立即升级”即可完成升级。升级前系统会自动备份代码和数据库，更新包和清单统一放在演示平台 `/updates/` 目录下。

## 功能模块

- 管理员登录、会员管理、手机号绑定与解绑。
- 设备列表、设备分组、设备状态、远程开门。
- 钥匙授权、钥匙领取、钥匙分享。
- 门禁卡、密码、指纹、人脸凭证管理。
- 区域、楼栋、单元、房号、房间绑定申请。
- 摄像头配置、查询和 Web 控制台。
- 操作日志、开门记录、系统配置、应用配置。

## 数据库

开源版本只保留一个最终 SQL 文件：

```text
weimenjin_open.sql
```

新库直接导入该文件即可。后续结构变更也直接合并到这个最终 SQL 文件中。
